# 国際化

```json
// resources/js/locales/ja.json
{
  "mypage": {
    "title": "マイページ"
  }
}
```

```vue
<script setup lang="ts">
import { useI18n } from 'vue-i18n';
const { t, locale } = useI18n();
</script>

<template>
  <h1>{{ t('mypage.title') }}</h1>
  <button @click="locale = 'ja'">日本語</button>
  <button @click="locale = 'en'">English</button>
</template>
```
